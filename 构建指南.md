# 优雅音乐播放器构建指南

## 构建环境配置

### 1. 安装JDK

确保已安装JDK 8或更高版本，并正确配置JAVA_HOME环境变量。

### 2. 安装Maven

已验证系统安装了Maven 3.9.9版本，可以通过以下命令检查：

```
mvn --version
```

### 3. 配置Android SDK

需要设置ANDROID_HOME环境变量，指向Android SDK安装路径。

在Windows系统中，可以通过以下步骤设置：

1. 右键点击"此电脑"，选择"属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
